## Getting started

#### Clone repository
```
  git clone https://github.com/jcagz96/ignite-lab-design-system.git

  cd ignite-lab-design-system
```

#### Install dependecies
```
  npm install
```

#### Run project in localhost
```
  npm run dev
```

#### Run storybook
```
  npm run storybook
```
